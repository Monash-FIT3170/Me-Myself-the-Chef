# Backend

### Instructions

1. Navigate to this folder
   ```
   cd backend
   ```
2. Install dependencies
   ```
   npm install
   ```
3. Create `.env` file in this folder
   ```
   PORT=
   RAPID_API_KEY=
   MONGO_DB_URI=
   ```
4. Start the server
   ```
   npm run dev
   ```
5. Go to `localhost:port` to see the page

6. Change the upstream branch (optional)
   ```
   git branch --set-upstream-to origin/branch
   ```
